# Abhishek Kumar — Portfolio

Simple, fast portfolio built with **React + Vite + CSS** (no TanStack, no MUI, no backend).

## Quick start

```bash
cd abhi-portfolio
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Deploy on Vercel

1. Push this folder to GitHub (e.g. `abhi-portfolio` repo).
2. Go to [vercel.com](https://vercel.com) → **Add New Project** → import the repo.
3. Vercel auto-detects Vite. Defaults:
   - **Build command:** `npm run build`
   - **Output directory:** `dist`
4. Click **Deploy**.

No extra config required.

## Scripts

| Command         | Description              |
| --------------- | ------------------------ |
| `npm run dev`   | Local dev server         |
| `npm run build` | Production build to `dist` |
| `npm run preview` | Preview production build |

## Customize

Edit `src/data/content.js` for skills, experience, projects, and links.
